About Disease Recognition and Test Recommendation System

Advanced ML-based disease recognition and test recommendation

Disease Recognition and Test Recommendation System is an intelligent healthcare application that uses machine learning to analyze patient symptoms, predict probable diseases with confidence scores, and recommend appropriate medical tests to support early and informed clinical decision-making.

Interpretable ML

Feature importance and SHAP explanations provide transparency into how predictions are made, giving clinicians confidence in the output.

Detailed Inputs

Symptom severity & duration are integrated into the model inputs for richer, more contextually accurate predictions.

Smart Recommendations

Dynamic test recommendations are generated based on confidence levels and symptom patterns unique to each prediction.

How It Works
01
Enter Symptoms

Select symptoms from a library of 132+ clinical symptoms. Optionally add severity (1–5) and duration in days for each one.

02
ML Analysis

Our ensemble model processes your inputs, generates a confidence-weighted disease prediction, and ranks contributing symptoms.

03
Review & Report

Review the predicted disease, top symptoms, and recommended lab tests. Download a printable PDF report if logged in.
